# MovieFetch (myFlix Frontend)

This project is the client-side for the MovieFetch app built with React.

## Tech Used

- React
- SCSS
- Parcel

## Setup

To run the app locally:

1. Clone the repo
2. Run `npm install`
3. Run `parcel src/index.html`
4. Open `http://localhost:1234`
