Movie Fetch

This API provides access to movie data. Use it to search, save, and explore films from your collection.

API Endpoints

Request URL Response
GET /movies Return a list of all movies
GET /movies/:title Return data about a single movie by title
GET /genres/:genreName Return description of a genre by name
GET /directors/:directorName Return data about a director by name
POST /users Register a new user
PUT /users/:username Update user info
POST /users/:username/movies/:movieId Add a movie to user's favorites
DELETE /users/:username/movies/:movieId Remove a movie from user's favorites
DELETE /users/:username Deregister an existing user

Endpoint Details

  • GET /movies
    Returns an array of all movies with full details.
    Params: None
    Response: Array of movie objects (title, description, genre, director, imageURL)
  • GET /movies/:title
    Returns full details of a specific movie by title.
    Params: :title
    Response: Movie object (title, description, genre, director, imageURL)
  • GET /genres/:genreName
    Returns description of a genre.
    Params: :genreName
    Response: Genre description string
  • GET /directors/:directorName
    Returns director information.
    Params: :directorName
    Response: Director info string
  • POST /users
    Registers a new user.
    Body: { "username": "sallydev" }
    Response: User registered confirmation string
  • PUT /users/:username
    Updates a user's username.
    Params: :username
    Body: { "username": "newName" }
    Response: Update confirmation string
  • POST /users/:username/movies/:movieId
    Adds a favorite movie.
    Params: :username, :movieId
    Response: Movie added confirmation string
  • DELETE /users/:username/movies/:movieId
    Removes a movie from favorites.
    Params: :username, :movieId
    Response: Movie removed confirmation string
  • DELETE /users/:username
    Deregisters a user.
    Params: :username
    Response: Deregistration confirmation string
