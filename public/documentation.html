Movie Fetch

This API provides access to movie data. Use it to search, save, and explore films from your collection.

API Endpoints

Request URL Response
GET /movies Return a list of all movies
GET /movies/:title Return data about a single movie by title
GET /genres/:genreName Return description of a genre by name
GET /directors/:directorName Return data about a director by name
POST /users Register a new user
PUT /users/:username Update user info
POST /users/:username/movies/:movieId Add a movie to user’s favorites
DELETE /users/:username/movies/:movieId Remove a movie from user’s favorites
DELETE /users/:username Deregister an existing user
